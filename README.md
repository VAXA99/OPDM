# OPDM (Occupational Pathology Document Manager) / ПРОФПАК

Программа для автоматизации документооборота в отделениях профпатологии и медицинских учреждениях.

---

## Описание

**OPDM** (на русском — **ПРОФПАК**) — это простое в использовании десктопное приложение для массовой генерации медицинских документов на основе шаблонов Word и Excel-таблиц.  
Приложение предназначено для врачей, лаборантов, сотрудников отделов профпатологии и профмедицины.

- Быстро формирует персональные документы по шаблону для десятков/сотен сотрудников.
- Сохраняет файлы напрямую на диск или одним ZIP-архивом.
- Не требует интернета. Работает на любой Windows-машине с правами пользователя.

---

## Ключевые функции

- **Автоматическое создание Word-документов** на основании данных из Excel.
- **Гибкое сохранение**: DOCX-файлы по отдельности или всё вместе в ZIP-архив.
- **Полная локальность**: все файлы хранятся и создаются только на вашем компьютере.
- **Настраиваемые шаблоны**: поддержка любых ваших Word-шаблонов.
- **Интуитивно понятный интерфейс**: выбор папок, подпапок, имён архивов, всех путей — прямо в программе.

---

## Быстрый старт

1. **Запустите OPDM/ПРОФПАК** (двойной клик по EXE).
2. **Введите место работы** — будет подставлено в каждый документ.
3. **Выберите, что сохранять**:
    - DOCX-файлы (каждый документ отдельным файлом в выбранную папку)
    - ZIP-архив (все документы в одном архиве)
    - Можно выбрать оба варианта одновременно
4. **Для каждого варианта укажите путь сохранения и подпапку/имя архива**.
5. **Загрузите Excel-файл с данными** (список сотрудников).
6. **Загрузите Word-шаблон** (пример документа).
7. **Нажмите "Начать генерацию"** — готовые файлы окажутся в выбранных папках.

---

## Требования

- Windows 10/11 (x64)
- Не требует установки Python, работает "из коробки"
- Запуск с правами пользователя (администратор не требуется)

---

## Формат Excel

Excel-файл должен содержать столбцы:
- ФИО сотрудника
- Дата рождения
- Должность
- Факторы риска
- Диагноз (МКБ-10)

Шапка таблицы может быть произвольной — программа сама определяет расположение нужных столбцов.

---

## Безопасность

- Программа **не использует интернет**.
- Все персональные данные и документы остаются только на вашем ПК.
- Нет скрытых обновлений и телеметрии.
